## MySQL All for one

![MySQL](https://logowik.com/content/uploads/images/301_docker.jpg)

### Proposta Apresentada: 

Uma das principais vantagens de usar o Docker é a potencial escalabilidade que ele proporciona para execução das nossas aplicações, e quando falamos em escala, podemos pensar numa relação de quantidade vs. qualidade.

Nesse projeto foi utilizado o Dockerfile e o Docker-Compose, para a criação de volumes e criação de Containers multi-Stage, para rodar imagens de diversos Sistemas Operacionais do frontend, backend.
